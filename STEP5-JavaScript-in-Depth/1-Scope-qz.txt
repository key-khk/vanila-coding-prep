Q1. Line 8에서 출력되는 값은?  답 : ③ 4
① 0   ② 5   ③ 4   ④ undefined   ⑤ Error
----------------------------------------------------------------------------------
⑴| function foo () {
⑵|   var a = 5;
⑶|
⑷|   for (var i = 0; i < a; i++) {
⑸|     console.log(a);
⑹|   }
⑺|
⑻|   console.log(i); // 무엇이 출력될까요?
⑼| }
⑽|
⑾| foo();
----------------------------------------------------------------------------------
❶ Line 1~6은 foo함수 생성
❷ Line 11 foo 에서 함수 실행
❸ a라는 함수 변수 선언 후 5라는 값을 할당
❹ Line 4에서 반복문인 for문 생성, i는 0, a의 값인 5보다 커질때까지 +1씩 반복
❺ Line 5에서 a의 값을 출력하는 구문을 i<a가 될때까지 반복해서 실행 = 55555 출력
❻ Line 8에서 i 변수를 출력하는 구문이 실행 = 5 출력
❼ Line 11에서 foo 함수 종료 및 실행문의 종료를 끝으로 프로그램도 종료

Q2. Line 6에서 출력되는 값은? 답 : ① 1
① 1   ② undefined   ③ Error
----------------------------------------------------------------------------------
⑴| var a = 1;
⑵| 
⑶| function bar () {
⑷| 
⑸|   function foo() {
⑹|     console.log(a); // ?
⑺|   }
⑻| 
⑼|   foo();
⑽| }
⑾| 
⑿| bar();
---------------------------------------------------------------------------------
❶ Line 1에서 a의 값이 1이라고 선언
❷ Line 3~10까지는 bar 함수 생성 및 foo함수의 값을 가진다.
❸ bar 함수 안에 Line 5~7까지 foo 함수 생성
❹ Line 6, foo 함수는 a의 값을 가진다.
❺ Line 9 foo함수 종료 및 foo함수의 값인 a를 bar함수 값으로 가지게 한다.
❻ Line 12 bar 함수 종료 및 실행문의 종료를 끝으로 프로그램도 종료

Q3. Line 6에서 출력되는 값은? 답 : ① 1
① 1   ② 2   ③ undefined   ④ Error
----------------------------------------------------------------------------------
⑴| var a = 1;
⑵| 
⑶| function bar () {
⑷| 
⑸|   function foo() {
⑹|     console.log(a);   // ?
⑺|   }
⑻| 
⑼|   a = 2;
⑽| 
⑾|   foo();
⑿| }
⒀| 
⒁| bar();
----------------------------------------------------------------------------------
❶ Line 1에서 a의 값이 1이라고 선언
❷ Line 3~12까지는 bar 함수 생성 및 foo함수의 값을 가진다
❸ bar 함수 안에 Line 5~7까지 foo 함수 생성
❹ Line 6, foo 함수는 a(=1)의 값을 가진다.
❺ Line 9에서 a = 2---->만일 Line9가 없다면 Line1에서 선언한 a=1의 값을 가진다.
❻ Line 11에서 foo함수 종료 및 foo함수의 값인 a를 bar함수 값으로 가지게 한다.
❼  Line 14 bar 함수 종료 및 실행문의 종료를 끝으로 프로그램도 종료

Q4. Line 8에서 출력되는 값은? 답 : 
① 1   ② 2   ③ undefined   ④ Error
----------------------------------------------------------------------------------
⑴| function foo () {
⑵|   var a = 1;
⑶| 
⑷|   function bar () {
⑸|     a = 2;
⑹|   }
⑺| 
⑻|   console.log(a); // ?
⑼|   bar();
⑽| }
⑾| 
⑿| foo();
----------------------------------------------------------------------------------
❶ Line 1~10까지 foo 함수 생성
❷ foo 함수 안에 Line 4~6까지 bar 함수 생성
❸ Line 5, bar 함수 안에서 a=2라는 내용을 가지게 된다.
❹ Line 8에서 a를 출력( foo 함수는 Line 1에서 선언한 a의 값을 출력한다. )
----> '❸'에서 말한 값은 bar의 내용물인 a=2값이지 foo의 a 값은 될 수 없다.
❺ Line 11에서 bar함수 종료
❻ Line 12에서 foo함수 종료 및 실행문의 종료를 끝으로 프로그램도 종료

Q5. Line 8에서 출력되는 값은? 답 : ③ undefined
① 1   ② 2   ③ undefined   ④ Error
----------------------------------------------------------------------------------
⑴| var x = 1;
⑵| 
⑶| function foo () { 
⑷|     if (x > 1) {
⑸|         var x = 2;
⑹|     } 
⑺| 
⑻|     console.log(x);
⑼| } 
⑽| 
⑾| foo();
----------------------------------------------------------------------------------
❶ Line 1에서 x의 값이 1이라고 선언
❷ Line 3~9까지 foo 함수 생성
❸ Line 4에서 if문 조건의 값은 false
❹ Line 5는 if문이 true일때 사용, false일때 출력할 수 있는게 정의되지 않음.
❺ Line 8 false일때의 정의가 없으므로 undefined 출력
❻ Line 11에서 foo함수 종료 및 실행문의 종료를 끝으로 프로그램도 종료
